INTRODUCTION

The social media project is a coding assignment for Trinity College Dublin.

The project challenges me to use Express.js with as a backend and Ejs as a frontend to familiarize myself with a JavaScript Full stack Web.

Note that the images in the footage are not owned by me and only used for educational purposes.

The website used the following libaries for the following functionalities:



1. Passport – For logins.

2. Accessibility both for Disability and Hardwares.

3. Validator – Validation!

4. Sharp – To resize uploaded images for the homepage.
